# Welbodi na Gentri

**Welbodi na Gentri** means "Health is Wealth" and is a professional health information website for Sierra Leone.
This project is built with pure HTML, CSS, and JavaScript, optimized for mobile-first layouts and low-bandwidth environments.

## Getting started

Open `index.html` in your browser to view the homepage. Use the files in `pages/` for secondary sections and `assets/` for styling and scripts.

## Notes

- All files are starter files with boilerplate content.
- The site is intended to be fully responsive and suitable for low-bandwidth users.
- Add image assets to `assets/images/`.
